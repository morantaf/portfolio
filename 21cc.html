21cc app - server side

Helping logistics workers, one training at a time

Project Overview

Helping people with tech is exactly what this project is about. In the app, logistics worker from all over the world are able to access modules, learning games and many different medias in order to upskill themselves and to be able to apply for better jobs.

In a 3 developers team (1 team lead, 1 Frontend and 1 Backend developer), I was in charge of implementing the REST API to communicate with the Frontend application. I also did some frontend work using the inbuilt language of the Grails framework. Very similar to Spring MVC and its jsp, it supercharge development and makes creating pages easy, even for a backender :). It was mostly to create pages for the admin portal. The project was developed in Groovy using the Grails framework

Some major accomplishment I achieved on in this project:

  • • An analytics tool to allow admins to track the behaviour of the users on the app and to render a summary in PDF.
  • • An internal messaging functionality, giving the option to companies to send a message to their employees in the app
  • • A firewall against brute force attack
  • • Improving the response time of the server, making it possible to handle 4 times more concurrent users without the server slowing down

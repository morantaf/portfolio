VolkerWessels' portal - server side

Helping a leading construction company to achieve its goals

Project Overview

Volkerwessels is a well-known construction company in the Netherlands. Their internal webshop, developed in partnership with Geckotech, offers company an easy way to order and manage building materials for their construction projects.

Working in sprint with 2 other developers (1 team lead and 1 Frontend developer), I was taking care of most of the backend workload. I've implemented a lot of new funcionalities, created a new api for one of their new interfaces, fixed countless bugs, created new pages for the admins' backoffice and the list goes on. The project was developed in Groovy using the Grails framework

Some major achievments accomplished on in this project:

  • • Upgraded the Elasticsearch engine to the latest version.
  • • Refactored all the unit tests to make them self contained
  • • Developed an API for a new interface

If you want to read more about Geckotech, they're an amazing company: https://geckotech.nl/

If you want to read more about Volkerwessels' portal: https://www.vwml.nl/nl/vwml-portal

Good to know about Groovy and Grails

Apache Groovy is a Java-syntax-compatible object-oriented programming language for the Java platform. It is both a static and dynamic language with features similar to those of Python, Ruby, and Smalltalk. Source: Wikipedia

Grails is built on top of and is part of the Java platform meaning that it is very easy to integrate with Java libraries, frameworks and existing code bases. Source: Wikipedia

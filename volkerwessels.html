VolkerWessels' portal - server side

Helping a leading construction company to achieve its goals

Project Overview

Volkerwessels is one of the most well known construction company. Their internal webshop, developed in partnership with Geckotech, offers an easy way to order and manage building materials for construction projects.

Working in sprint with 2 other developers (1 team lead who was also a backender and 1 Frontend developer), I was taking care of most of the backend workload. I've implemented a lot of new funcionalities, created a new api for one of their new interfaces, fixed countless bugs, created new pages for the admins' backoffice and the list goes on.

Some major achievments accomplished on in this project:

  • • Upgraded the Elasticsearch engine to the latest version.
  • • Refactored all the unit tests to make them self contained
  • • Developed a new API for a new interface

If you want to read more about Geckotech, they're an amazing company: https://geckotech.nl/

If you want to read more about Volkerwessels' portal: https://www.vwml.nl/nl/vwml-portal

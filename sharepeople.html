Sharepeople's portal - server side

Disrupting the insurance industry

Project Overview

The concept behind Sharepeople is very clever, and they couldn't have asked for a better partner than Geckotech to tackle the task to disrupt the insurance industry for entrepreneur. The idea is that when a member is sick, other members jump in to help him cover for his incapacited period

I was the backend developer in a 3 persons team (1 team lead, 1 Backend and 1 Frontend developer) and was tasked to maintain and improve the REST API for the portal and their onboarding platform. I also had to write a lot of unit and integration tests. Finally, I was responsible for the release of the server side application. The project was developed in Groovy using the Grails framework

Some major achievments accomplished on in this project:

  • • A firewall to protect the back-office.
  • • A new API for the onboarding process of their most recent product Sharepeople Lang
  • • An algorithm to calculate the validity of a BSN number
  • • Improving the integration with a 3rd party e-mail marketing platform

If you want to read more about Geckotech, they're an amazing company: https://geckotech.nl/

If you want to read more about Sharepeople and their products: https://www.sharepeople.nl/
